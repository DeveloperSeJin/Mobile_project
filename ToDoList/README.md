# ToDoList
------------
## 프로젝트 요약
> 하루 목표 및 한달 목표를 설정할 수 있는 ToDoList 모바일 어플리케이션

## 기능설명
------------
1. 빈칸에 해야할 일 작성
2. 빈칸 상단에 clik on your schedule 클릭한 후 목표시간 설정
3. 중요한 일은 importance 버튼을 눌러 활성화
4. add를 눌러 목표 등록
5. 월간 목표는 좌측 상단의 바를 눌러 달력으로 이동
6. 달력의 빈칸에 목표 입력
7. add를 누르면 월간 목표 등록 완료
